# App Store Screenshot Checklist

Capture on **iPhone 15 Pro Max** simulator (6.7") or physical device. Sign in with the **reviewer demo account** before capturing.

## Required screens (minimum 3–5)

| # | Screen | What to show |
|---|--------|----------------|
| 1 | Home (signed in) | Daily mood check-in + Book Appointment banner |
| 2 | Book | In-app booking form with service/date/time selected |
| 3 | Events | RSVP toggles on event cards |
| 4 | Profile | Account info + Privacy Policy link + Delete account button |
| 5 | Membership | Plan status / entitlements (no purchase buttons on iOS) |

## Dimensions (Apple 6.7" display)

- Portrait: **1290 × 2796 px** (or 1284 × 2778)

## Do not use

- Guest-only / logged-out home as primary screenshots
- Service directory pages with only external website links
- Placeholder Expo splash or default icons

## After capture

1. Upload to App Store Connect → App Store → Screenshots → iPhone 6.7"
2. Match promotional text and description in [APP_STORE_CONNECT_SUBMISSION_GUIDE.md](../APP_STORE_CONNECT_SUBMISSION_GUIDE.md)
3. Paste review notes from [APPLE_APP_REVIEW_ANALYSIS.md](APPLE_APP_REVIEW_ANALYSIS.md)
